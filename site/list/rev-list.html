Reverse List

Given an input list, reverse the list in place.

Sample Input:

[1, 2, 3]

Sample Output:

[3, 2, 1]

Complexity:

O(n) - space

O(1) - time

Python Code:


def rev_list(arr):
    left = 0
    right = len(arr) - 1

    while left < right:
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1
    
    return arr

arr = [1, 2, 3]
print(rev_list(arr))

Output:

[3, 2, 1]
