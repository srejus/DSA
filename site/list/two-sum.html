Two Sum Problem

Given a list of integers and a target sum, find two distinct elements in the list that add up to the target sum.

Sample Input:

[2, 7, 11, 15], target = 9

Sample Output:

[0, 1] (since 2 + 7 = 9)

Complexity:

O(n) using a dictionary

Python Code:


def two_sum(arr, k):
    seen = set()
    for i in arr:
        if k - i in seen:
            return True
        seen.add(i)
    
    return False

arr = [2, 7, 11, 15]
k = 9
x = two_sum(arr, k)
print(x)

Output:

True
