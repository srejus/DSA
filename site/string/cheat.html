String CheatSheet

1. String Creation

s1 = "Hello"
s2 = 'World'
s3 = """Multi-line
String"""

2. Common Methods

Operation Syntax Example
Length len(s) len("Hello") → 5
Lowercase s.lower() "HeLLo".lower() → "hello"
Uppercase s.upper() "hello".upper() → "HELLO"
Title Case s.title() "hello world".title() → "Hello World"
Strip Whitespaces s.strip() " Hello ".strip() → "Hello"
Replace Substring s.replace(old, new) "Hello World".replace("World", "Python") → "Hello Python"
Find Substring Index s.find(sub) "Hello".find("l") → 2
Check Start s.startswith(sub) "Hello".startswith("He") → True
Check End s.endswith(sub) "Hello".endswith("o") → True

3. Accessing and Slicing

s = "Hello World"
Access: s[0] → 'H'
Slice: s[0:5] → "Hello"
Reverse: s[::-1] → "dlroW olleH"

4. Concatenation and Repetition

s1 = "Hello"
s2 = "World"
Concatenate: s1 + s2 → "HelloWorld"
Add space: s1 + " " + s2 → "Hello World"
Repeat: s1 * 3 → "HelloHelloHello"

5. Split and Join

Operation Syntax Example
Split into list s.split() "a b c".split() → ['a', 'b', 'c']
Split by delimiter s.split(delim) "a,b,c".split(",") → ['a', 'b', 'c']
Join list into string delimiter.join(list) ",".join(['a', 'b', 'c']) → "a,b,c"

6. Checking Conditions

Operation Syntax Example
Alphabetic s.isalpha() "Hello".isalpha() → True
Numeric s.isdigit() "123".isdigit() → True
Alphanumeric s.isalnum() "Hello123".isalnum() → True
Lowercase s.islower() "hello".islower() → True
Uppercase s.isupper() "HELLO".isupper() → True

7. Formatting Strings

name = "Alice"
age = 25
Old: "Name: %s, Age: %d" % (name, age)
Format: "Name: {}, Age: {}".format(name, age)
f-strings: f"Name: {name}, Age: {age}"
