Count All Subsequences

Given an array of integers, count all possible subsequences.

Python Code:


def count_subsequences(arr, ind=0):
    if ind >= len(arr):
        return 1  # There's exactly one subsequence when all elements are considered
    
    include_current = count_subsequences(arr, ind + 1)  # Count subsequences including current element
    exclude_current = count_subsequences(arr, ind + 1)  # Count subsequences excluding current element
    
    return include_current + exclude_current

arr = [4, 1, 3]
result = count_subsequences(arr)
print("Total number of subsequences:", result)

Output:

Total number of subsequences: 8
